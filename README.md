
# Legal-Rights-Assistance Website — Landing Page (Vite + React + Tailwind)

Landing site for **Legal-Rights-Assistance** linking to the interactive Streamlit demo. Built with Vite, React (TypeScript), and Tailwind CSS. Deploy to Vercel for fast global hosting.

Live demo (landing): `https://legal-rights-assistance.vercel.app/`  
Streamlit backend demo: (link from landing page)

---

## Features
- Hero with clear CTA to the Streamlit app
- Problem / Impact / How it works / Features / Supported domains sections
- Responsive layout and lightweight animations
- "Ask Query" button opens the Streamlit app in a new tab
- Easy to customize and redeploy

## Tech stack
- Vite
- React + TypeScript
- Tailwind CSS
- Lucide icons (or Heroicons)
